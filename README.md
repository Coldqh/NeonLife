# NEON LIFE — v0.4.0

Вертикальный системный срез **SEVEN DAYS BELOW**.

## Уже работает

- адаптивная оболочка desktop/mobile;
- компактный мобильный LIFE под iPhone 14 Pro;
- отдельный канон NEON LIFE без данных из других игровых сессий;
- детерминированная генерация города, трёх районов, организаций, локаций, героя и главного контакта;
- единый `WorldState` со стабильными Entity ID;
- три независимых слота сохранения в IndexedDB;
- автосохранение и ручное сохранение;
- создание нового мира в любом слоте;
- миграция старой localStorage-сессии с сохранением только механических параметров;
- проверка контрольной суммы сохранения;
- архив повреждённых записей и безопасное создание нового мира;
- очередь будущих событий;
- детерминированный WORLD PULSE активного района;
- изменение безопасности, полиции, давления улицы, транспорта, рынка и энергосети;
- быстрые действия с длительностью, стоимостью и последствиями;
- журнал событий, окна, контекст NPC и системная диагностика;
- проверка версии и принудительное PWA-обновление;
- GitHub Pages workflow.

## Запуск

```bash
npm install
npm run dev
```

## Проверка production-сборки
