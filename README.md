# NEON LIFE — v0.5.0

Вертикальный системный срез **SEVEN DAYS BELOW**.

## Уже работает

- адаптивная оболочка desktop/mobile;
- компактный мобильный LIFE под iPhone 14 Pro;
- отдельный канон NEON LIFE без данных из других игровых сессий;
- детерминированная генерация города, трёх районов, организаций, локаций, героя и главного контакта;
- единый `WorldState` со стабильными Entity ID;
- три независимых слота сохранения в IndexedDB;
- автосохранение, миграции, checksum и recovery;
- очередь будущих событий и WORLD PULSE активного района;
- проверка версии и принудительное PWA-обновление;
- физическое положение героя в конкретной локации;
- маршруты между объектами с временем, стоимостью и задержками транспорта;
- расписания открытия магазинов, мастерской и офисов;
- отдельные продукты собственного лора NEON LIFE;
- домашний пищевой запас, количество, сроки хранения и порча;
- локальные магазины с ограниченным остатком товара;
- доставка продуктов в домашний пищевой шкаф;
- требования к приготовлению: нагреватель, горячая вода, кухня и пищевой принтер;
- сон дома с восстановлением, влиянием шума и качества жилья;
- компактные модули CITY, INVENTORY и HOME;
- журнал событий, окна, контекст NPC и системная диагностика;
- GitHub Pages workflow.

## Запуск

```bash
npm install
npm run dev
```

## Проверка production-сборки

```bash
npm run build

```
