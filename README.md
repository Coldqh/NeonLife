# NEON LIFE — v0.3.0

Вертикальный системный срез **SEVEN DAYS BELOW**.

## Уже работает

- адаптивная оболочка desktop/mobile;
- компактный мобильный LIFE под iPhone 14 Pro;
- экран LIFE с состоянием героя, планом, возможностями и журналом;
- ручное продвижение игрового времени;
- детерминированный WORLD PULSE для Sector 04;
- изменение безопасности, полиции, давления банды, транспорта, рынка и энергосети;
- локальные события, возникающие из состояния района;
- централизованный seeded random и стабильные Entity ID;
- быстрые действия с длительностью, стоимостью и последствиями;
- контекстная панель NPC;
- внутренняя оконная система;
- настройки интерфейса с сохранением в localStorage;
- проверка версии через `version.json`;
- блокирующее принудительное обновление устаревшей сборки;
- автоматическая смена service worker и очистка старых PWA-кэшей;
- GitHub Pages workflow.

## Запуск

```bash
npm install
npm run dev
```

## Проверка production-сборки

```bash
npm run build
npm run preview
```
