# NEON LIFE v0.30.0 — MOBILE VISUAL SYSTEM

## Цель

Полностью заменить старую мобильную админ-панель на цельный игровой интерфейс под iPhone 14 Pro, не меняя симуляцию и игровые правила.

## Добавлено

- новая мобильная оболочка `MobileExperience`;
- главная, профиль, глобальная карта, локальная карта, Nearby и перемещение;
- единая нижняя навигация с одним активным пунктом;
- профессия игрока определяется состоянием мира и не фиксируется как «курьер»;
- глобальная карта рисует полный массив секторов города;
- дорожные связи, границы районов, транспортные станции и нагрузка выводятся поверх сетки;
- выбранный сектор раскрывается в локальную карту улиц и кварталов;
- локальная карта использует реальные здания, локации и физическую позицию игрока;
- Nearby использует физических NPC, здания, автомобили и события активного сектора;
- сохранены существующие действия входа в здания, посадки в машины и перемещения;
- добавлены локальные WebP-ассеты для портретов, мест и транспорта;
- добавлена автоматическая проверка визуальных инвариантов.

## Визуальная система

- один шрифт во всём мобильном приложении;
- общий отступ 20 px;
- сетка 4 px;
- page title `28/34 700`;
- section title `18/24 650`;
- body `15/21 400`;
- caption `13/18 400`;
- numeric value `22/28 650`;
- радиус обычной карточки не больше 16 px;
- единая navy-black / charcoal / red палитра;
- зелёный и янтарный используются только для статусов;
- без декоративных карточек внутри карточек и без дублирования баланса.

## Проверки

1. TypeScript приложения прошёл с локальными декларациями React, соответствующими публичным типам проекта.
2. Все 12 доменных TypeScript-конфигураций прошли без ошибок.
3. `npm run test:mobile-ui`: 35/35 архитектурных и визуальных инвариантов.
4. Chromium visual audit: 40/40 проверок четырёх ключевых экранов.
5. Responsive audit: 16/16 рендеров на ширинах 375, 390, 393 и 430 px без горизонтального overflow, обрезанного текста и двойной активной навигации.
6. Проведено три последовательных визуальных прохода: исправлены переполнение профиля, несогласованная сетка карты, нечитаемая легенда, плотность Nearby, фиксированный bottom sheet, адаптивная типографика и ещё более двадцати заметных расхождений.

Полный `npm run build` в локальной среде не запускался: npm-зависимости отсутствуют, а доступ к npm registry закрыт. Исходники и все доступные TypeScript-конфигурации проверены локально.
