# NEON LIFE v0.10.0 — CITY SITUATIONS / DECISION PAUSE

Патч добавляет причинные ситуации, которые останавливают время и заставляют игрока принимать конкретное решение внутри уже работающих систем.

## Главное

- блокирующая пауза при важном событии;
- проверка сомнительного курьерского груза;
- спор с клиентом после опоздания или повреждения;
- помощь разваливающейся рабочей смене;
- личный разговор с NPC под высоким давлением;
- последствия для денег, времени, рейтинга, бизнеса, памяти и отношений;
- история решений;
- схема сохранений 8;
- PWA 0.10.0.
